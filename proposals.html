A digital platform for dealerships to manage car inventory, test drives, and customer requests efficiently.

Problem & Target Audience

Many small and mid-sized car dealerships still rely on manual spreadsheets and outdated CRM systems. AutoLink targets these dealerships by offering a modern, cloud-based management system that connects sales staff, inventory, and customers seamlessly in one place.

Core Features (MVP)

  • User authentication (Admin and Customer roles)
  • Car catalog with filtering and search
  • Test drive & order request forms with validation
  • Admin dashboard with CRUD for cars and requests
  • Automated email + Google Calendar integration via n8n
  • Database management for Users, Cars, and Requests
  • Responsive, modern UI built with Vue.js & Tailwind
